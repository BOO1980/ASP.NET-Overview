Creating Your First ASP.NET Web site (Visual Basic)

  1. Open Visual Studio
  2. Click File Menu and Choose New Project
  3. Select Visual Basic
  4. Select Web
  5. Select Previous Versions
  6. Select ASP.NET Web Forms Site
  7. Select .NET Framework from the drop down list
  8. Change the location on the disk where you want the Web site stored
  9. Open the file Default.aspx by double clicking it
  10. Remove all the code inside the <asp:Content> block that has its ID set to BodyContent (it starts with <div> at line 5 and ends with </div>) all the way to the bottom.
  11. Replace it with the following bolded code

    <h2>Hello World </h2>

    <p>Welcome to Beginning ASP.NET 4.5.1 on <%: DateTime.Now.ToString() %></p>

  12. You will notice a small icon representing IIS Express, this is a web server that has been started by VS automatically to serve the request for the page

How it Works

Although the website is simple, the process that eventually results in the page Default.aspx being displayed in your browser isn't so simple. ASP.BET page can't do much. It needs to be processed and served by a web server before the browser can display it. Thats why VS automatically started up by IIS Express to handle the request for the page

It is important to realize that the ASPX file you modified in VS is not the same as the one that eventually gets displayed by the browser

When you create a page in VS you add markup to it. The markup is an aspx page that is a combination of HTML code for ASP.NET Server controls, code written in Visual Basic.NET or C# and more

When you request as ASPX page in your browser the web server process the page, executes any server-side code it finds in the file, and effectiveness transforms the ASP.NET markup into plain HTML that it then sends to the browser, where it is displayed
