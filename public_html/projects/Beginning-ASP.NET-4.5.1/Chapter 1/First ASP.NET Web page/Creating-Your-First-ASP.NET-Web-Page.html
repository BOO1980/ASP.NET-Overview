Creating Your First ASP.NET Empty Web Page

  1. Start Visual Studio
  2. Choose File -> New -> Web Site, which triggers the New Web Site Dialog Box
  3. Make sure that ASP.NET Empty Web Site
  4. Right click the new web site in the solution explorer
  5. Choose Add -> Add New Item
  6. In the new window click Web Form and type ControlDemo
  7. Switch the page to Design View by clicking the design button at the bottom of the Document Window
  8. Click the Toolbox and click the pin icon to make it visible
  9. Drag a Textbox and a Button from the standard category onto the page
  10. Right click the button and choose properties
  11. In the properties grid locate the text property (under appearance) and change it from Button to Submit Information

How it works

You could of typed the below code to have the same effect

<asp:TextBox ID="TextBox1" runat="server"></asp:TextBox> <asp:Button ID="Button1" runat="server" Text="Submit Information" />

<asp:Button ID="Button1" runat="server" Text="Submit Information" />
